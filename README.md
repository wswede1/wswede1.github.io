# HLT Portfolio Website

This repository contains my professional portfolio website for the University of Arizona MS in Human Language Technology program.

## About This Site

This portfolio showcases my work in Natural Language Processing and Human Language Technology, including:
- Professional introduction and background
- Curriculum Vitae / Resume
- Detailed project descriptions demonstrating HLT competencies
- Code examples and technical documentation

## Projects Featured

1. **Internship Project** - RAG project using podcast transcripts and Ollama.
2. **LING 508 Course Project** - A yoga pose database.

## Built With

- [Jekyll](https://jekyllrb.com/) - Static site generator
- [Minimal Mistakes](https://mmistakes.github.io/minimal-mistakes/) - Jekyll theme
- [GitHub Pages](https://pages.github.com/) - Hosting
